# desktopmenangkap

> Akses informasi tentang sumber media yang dapat digunakan untuk menangkap audio dan video dari desktop menggunakan API [ ` navigator.mediaDevices.getUserMedia `].

Proses:[Renderer](../glossary.md#renderer-process)
